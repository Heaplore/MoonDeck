📐 设计要点
1. 统一骨架:卡片 = 圆角矩形 + 渐变底 + 月光边角光晕 + 标题栏(icon + name + 状态点) + 状态行(数据源 + 刷新时间)

3. 订阅主题:三张卡都从 ThemeManager 拿色值,切 dark/light/glass/silvermoon 都一致
4. 留白:内部 18px padding,行间 14px 节奏,避免视觉拥挤
5. 可扩展:之后做倒计时/便签/音乐等其他卡,直接继承骨架只换内容区
